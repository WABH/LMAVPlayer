# LMIJKPlayer
基于苹果原生AVPlayer，支持横屏、竖屏，上下滑动调节音量、屏幕亮度，左右滑动调节播放进度, 支持横竖屏UI不同的完整播放器Demo

## 特性
* 支持横、竖屏切换和展示不同UI
* 左侧1/2位置上下滑动调节屏幕亮度（模拟器调不了亮度，请在真机调试）
* 右侧1/2位置上下滑动调节音量（模拟器调不了音量，请在真机调试）
* 支持滑动屏幕快进快推
* 支持在当前页面播放新的视频
* 支持显示上次播放记录
* 支持从某个位置开始播放
* 支持播放页横竖屏, 其他页面仅支持竖屏

## 要求
* iOS8+
* Xcode8+
